XENO SHIELD - Alien Intrusion Detection System (Level 3 Industrial Project)

1. Project Overview
XENO SHIELD is an AI-powered Intrusion Detection System (IDS) designed to monitor, analyze, and detect suspicious network activities in real-time. 
The system uses a futuristic alien-themed interface to visualize cyber threats as unknown extraterrestrial signals. 
This project combines cybersecurity concepts, artificial intelligence, and modern web technologies to create a professional-grade security dashboard.

2. Project Objectives
- Monitor real-time network traffic
- Detect anomalies using AI/ML models
- Classify threats based on behavior patterns
- Provide real-time alerts
- Visualize attacks in an interactive dashboard
- Maintain logs for threat analysis

3. Core Features
- Real-time Packet Monitoring
- AI-based Anomaly Detection
- Threat Level Indicator (Low/Medium/High/Critical)
- Live Dashboard with Graphs
- Alert Notification System
- Attack Logs & History
- Suspicious IP Tracking
- Radar Visualization Interface

5. System Architecture
